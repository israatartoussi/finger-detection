# Finger Detection

This project demonstrates real-time finger detection using **OpenCV** and **MediaPipe** in Python.

## 🧠 Features

- Real-time hand and finger tracking
- Counts how many fingers are raised
- Lightweight and efficient
- Easy to run on any local machine
